# 📊 CodeAlpha Task 2: Unemployment Analysis with Python

## 📌 Project Overview
This project is submitted as **Task 2** for the **CodeAlpha Data Science Internship**. The objective is to analyze unemployment trends in India using Python by performing data cleaning, exploratory data analysis (EDA), and visualization to identify meaningful patterns, regional differences, and the impact of the COVID-19 pandemic.

---

## 🎯 Objectives
- Analyze unemployment trends in India.
- Clean and preprocess the dataset.
- Perform Exploratory Data Analysis (EDA).
- Visualize unemployment patterns across states and regions.
- Examine the impact of COVID-19 on unemployment rates.
- Generate actionable insights from the data.

---

## 🛠️ Tech Stack
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
The analysis uses unemployment data containing information such as:
- Region
- State
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)

---

## 📈 Analysis Performed
- Data Cleaning
- Missing Value Handling
- Date Formatting
- Feature Extraction
- Descriptive Statistics
- Unemployment Trend Over Time
- Top States by Unemployment Rate
- Urban vs Rural Unemployment Analysis
- COVID-19 Impact Analysis
- Data Visualization

---

## 📊 Visualizations
The project includes:
- Line Charts
- Bar Charts
- Histograms
- Box Plots
- Heatmap
- Count Plots

---

## 🔍 Key Insights
- Unemployment rates varied significantly across different states.
- Several regions consistently experienced higher unemployment than others.
- Urban and rural areas showed noticeable differences in unemployment trends.
- A significant rise in unemployment was observed during the COVID-19 period.
- Data visualizations revealed clear regional and temporal patterns in unemployment.

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/CodeAlpha_Unemployment_Analysis.git
```

2. Navigate to the project directory:

```bash
cd CodeAlpha_Unemployment_Analysis
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open **Task 2.ipynb** and run all cells.
